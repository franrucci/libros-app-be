# Backend - Reseñas de Libros App

Este es el backend de la aplicación (en proceso) de reseñas de libros, desarrollado con Node.js y Express. Proporciona APIs para gestionar libros, usuarios y autenticación.

---

## ⚙️ Tecnologías utilizadas

- Node.js  
- Express  
- MongoDB  
- Mongoose  
- Joi  
- Firebase  
- JWT
